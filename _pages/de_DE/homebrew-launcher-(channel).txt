---
title: "Homebrew Launcher (Channel)"
---

Hiermit wird Homebrew Channel installiert, so dass sobald du ein gepatchtes SysNAND am laufen hast, der Homebrew Launcher als Icon im Sytem-Menü angezeigt wird.

#### Anleitung

1. Enter the Homebrew Launcher
  + CBHC users must press (Home) while CBHC is booting to access the CBHC boot options menu, then select the Homebrew Launcher
  + Haxchi users must launch Haxchi once to patch SysNAND, then run it a second time while holding (A) to launch the Homebrew Launcher
  + Mocha CFW users must launch Mocha CFW, then relaunch the Homebrew Launcher
1. Starte WUP Installer GX2
  + Note that if WUP Installer GX2 crashes with a "DSi Exception Has Occurred" error, you should just shut down the console and try again.
1. Select the Homebrew Channel
1. Drücke "Install", dann drücke "Yes" zum bestätigen
1. Wähle "NAND" als Ziel
1. Select WUP Installer GX2
1. Drücke "Install", dann drücke "Yes" zum bestätigen
1. If you plan to install games to a USB HDD, select "USB" as the destination; if you do not plan to use a USB HDD, select "NAND" as the destination
1. Drücke (Home) um aus dem Installationsprogramm zu gelangen, sobald dieser fertig ist

___

Nun kannst du den Homebrew Launcher in SysNAND starten, indem du den Homebrew Channel startest.
{: .notice}

Jetzt kannst du den WUP Installer GX2 auf einem modifizierten SysNAND über das System Menü starten.
{: .notice}

Aus Sicherheitsgründen solltest du eine [Backup des NANDs](nand-backup) auf deinem Gerät erstellen.
{: .notice--primary}

Beachte, dass wenn diese DNS Adressen eingetragen sind, der Zugriff auf den eShop nur möglich ist, wenn zuerst NNU-Patcher mit Hilfe des Homebrew Launchers ausgeführt wurde (NNU-Patcher ist ein temporärer Patch und muss daher nach jedem Neustart neu ausgeführt werden).
{: .notice--info}

Beachte, dass wegen eines Bugs, der WUP Installer GX2 nur auf eine externe Festplatte installieren kann (z. B. auf eine HDD) wenn du WUP Installer GX2 auf deine USB-Festplatte mittels des Datenmanagement-Menüs der System-Optionen verschiebst.
{: .notice--warning}

Für Informationen wie du deine Discs in ein installierbares Format dumpen kannst, siehe [disc2app](disc2app).
{: .notice--success}

Für Informationen wie du vWii Mods in SysNAND installierst, siehe [vWii Modding](vwii-modding).
{: .notice--success}

Für Informationen wie du CFW sicher deinstallierst und auf Werkszustand zurückkehrst, siehe [CFW entfernen](uninstall-cfw).
{: .notice--warning}
