Esto instalará el Homebrew Channel para que, una vez que tengas una SysNAND modificada funcionando, el Homebrew Launcher aparezca como un icono en el menú del sistema.
{: .notice--info}

#### Instrucciones

1. Inicia el Homebrew Launcher usando `http://loadiine.ovh`
1. Inicia WUP Installer - Mod Y
1. Presiona (A) para instalar el Homebrew Channel
1. Presiona (Home) para salir del instalador una vez que haya completado la instalación

___

Ahora puedes iniciar el Homebrew Launcher en la SysNAND modificada simplemente iniciando el Homebrew Channel.
{: .notice}

Note that to access the eShop with the update blocking DNS addresses set, you must first run NNU-Patcher through the Homebrew Launcher (NNU-Patcher is a temporary patch and will need to be ran again if you need to access the eShop after a reboot).
{: .notice--info}

Para informarte acerca de cómo volcar los datos de tus discos de juego a un formato instalable para jugar sin necesidad del disco, echa un vistazo al apartado de [disc2app](disc2app).
{: .notice--success}

Para informarte acerca de cómo modificar la vWii en SysNAND, echa un vistazo al apartado de [Modear la vWii](vwii-modding).
{: .notice--success}

For information on uninstalling CFW safely and returning to stock, check out the [Uninstall CFW](uninstall-cfw) page.
