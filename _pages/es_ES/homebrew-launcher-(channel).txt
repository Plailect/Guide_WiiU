### Lectura requerida

This will install the Homebrew Channel so that, as long as Custom Firmware is being used, the Homebrew Launcher will appear as an icon on the system menu.

### Instrucciones

#### Section I - Installing the Homebrew Launcher Channel

1. Ingresa al Homebrew Launcher
  + Los usuarios de CBHC deben presionar (Home) mientras CBHC está iniciando para acceder al menú de opciones de inicio de CBHC y luego selecciona Homebrew Launcher
  + Haxchi users must launch Haxchi once to enable custom firmware, then run it a second time while holding (A) to launch the Homebrew Launcher
  + Mocha CFW users should already have entered this
1. Inicia WUP Installer GX2
  + Ten en cuenta que si WUP Installer GX2 crashea con el error "DSi Exception Has Ocurred", tan solo deberías apagar la consola y volver a intentar.
1. Selecciona el Homebrew Channel
1. Presiona "Install", luego presiona "Yes" para confirmar
1. Selecciona "NAND" como destino
1. Selecciona WUP Installer GX2
1. Presiona "Install", luego presiona "Yes" para confirmar
1. Selecciona "NAND" como destino
1. Presiona (Home), luego cierra el programa para salir del instalador una vez que haya terminado

#### Section II - Blocking System Updates

Note that, to access the eShop with these DNS addresses set, you must first run NNU-Patcher through the Homebrew Launcher. Additionally note that NNU-Patcher is a temporary patch and will need to be ran again if you need to access the eShop after a reboot.

Note that, unless you block updates with this method on all current and future connections, updates on the Wii U are performed *automatically* and *cannot* be disabled. 
{: .notice--danger}

1. Go to the "System Settings", then "Internet", then "Connect to the Internet" then press (X) to display your current network connections
1. For each of your connections (and all future connections), do the following
  + Select the connection
  + Select "Change Settings"
  + Move to the second list and choose "DNS"
  + Set the DNS to "Don't Auto-Obtain"
  + Set the DNS server IPs to the following servers
  + {:.bigdns} `168.235.092.108`
  + {:.bigdns} `081.004.127.020`
  + Select "Confirm", then press (B) to save
  + These servers will block your device from being updated

___

You can now launch the Homebrew Launcher on custom firmware just by launching the Homebrew Channel.
{: .notice--info}

You can now launch WUP Installer GX2 on custom firmware just by launching it from the system menu.
{: .notice--info}

Para mayor seguridad, *debes* crear un respaldo de tu consola siguiendo [Respaldo de la NAND](nand-backup).
{: .notice--primary}

Ten en cuenta que para acceder a la eShop con los las direcciones DNS que bloquean las actualizaciones, antes debes iniciar NNU-Patcher desde el Homebrew Launcher (NNU-Patcher es un parche temporal y tendrás que iniciarlo después de cada reinicio para entrar a la eShop).
{: .notice--info}

Ten en cuenta que, debido a un error, WUP Installer GX2 sólo puede realizar instalaciones en memorias USB (como un USB HDD) si mueves WUP Installer GX2 al almacenamiento USB usando el menú Gestión de datos de las Configuraciones de la consola.
{: .notice--warning}

Para informarte acerca de cómo volcar los datos de tus discos de juego a un formato instalable para jugar sin necesidad del disco, echa un vistazo al apartado de [disc2app](disc2app).
{: .notice--success}

For information on installing vWii mods on custom firmware, check out the [vWii Modding](vwii-modding) page.
{: .notice--success}

Para informarte acerca de cómo desinstalar el CFW de forma segura y regresar la consola a su estado original, echa un vistazo al apartado de [Desinstalar CFW](uninstall-cfw).
{: .notice--warning}
