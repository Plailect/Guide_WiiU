---
title: "Haxchi"
---

Haxchi est un custom firmware qui patche le contrôle des signatures, la vérifications de la région et exécute d'autres codes personnalisé en arrière-plan.
{: .notice}

Haxchi est un exploit qui permet de lancer le CFW directement depuis le menu d'accueil sans utiliser l'Homebrew Launcher.
{: .notice--info}

Si vous avez besoin d’acheter un des jeux exploitables, exécutez NNU-Patcher à partir de l'Homebrew Launcher puis lancer l’eShop. Une fois qu’il a été acheté, entrez à nouveau dans le [Homebrew Launcher](homebrew-launcher).
{: .notice--primary}


#### Prérequis

+ Une copie légitime de l’un des jeux Console Virtuelle DS suivants installé dans la mémoire interne de votre Wii U
  + **Ne pas** installer le jeu Console Virtuelle DS sur une clé USB
  + Actuellement, les titres les moins chers sur l'eShop (6,99 €) sont : " Programme d'entraînement cérébral du Dr Kawashima : Quel âge a votre cerveau ? ", " Cérébrale Académie ", " Pokémon Ranger : Nuit sur Almia ", " Pokémon Ranger : Sillages de lumière " et " Yoshi Touch & Go ".
  + Si vous aviez " Programme d'entraînement cérébral du Dr Kawashima : Quel âge a votre cerveau ? " installé sur votre Wii U depuis un certain temps, essayez de le supprimer et de réinstaller le jeu depuis l’eShop. Les anciennes versions du jeu ne fonctionnent pas avec Haxchi
  + Le jeu ne sera plus jouable après ce processus
  **Le jeu de console virtuelle DS que vous avez utilisé pour Haxchi DOIT être une copie légitime !**

#### Instructions

1. Lancez Haxchi Installer
1. Sélectionnez le jeu Console Virtuelle DS sur lequel vous avez installé Haxchi, puis appuyez sur (A)
1. Lisez les mises en gardes qui apparaîtront, puis appuyez sur (A) pour lancer l'installation
1. Une fois cette opération terminée, lancez votre jeu Console Virtuelle DS Haxchi (nommé " Haxchi ")
1. La console redémarrera dans la SysNAND patché
  + Ceci permettra aux titres non signés (par exemple la Chaîne Homebrew et backups de jeux) à être lancés directement depuis le menu Home

___

Actuellement, vous devez lancer le jeu DS Console Virtuelle à chaque fois que vous redémarrez la console pour entrer dans SysNAND patché. Dans la page suivante vous fera installer CBHC (Coldboot Haxchi) qui le fera automatiquement.
