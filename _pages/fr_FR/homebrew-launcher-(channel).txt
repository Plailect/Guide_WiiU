---
title: "Homebrew Launcher (Channel)"
---

Ceci va installera la Chaîne Homebrew pour que, quand votre SysNAND patché est exécuté, l'Homebrew Launcher apparaîtra sous forme d’icône dans le menu Home.
{: .notice--info}

#### Instructions

1. Lancez le Homebrew Launcher via `http://loadiine.ovh`, Haxchi, ou Coldboot Haxchi
  + Notez que les utilisateurs d'Haxchi doivent lancer Haxchi une fois pour patcher la SysNAND, ensuite lancez dans un second temps tout en maintenant (A) pour lancer l'Homebrew Launcher
1. **Utilisateurs de l'Homebrew Laucher Uniquement:** Lancez Mocha CFW, ensuite relancer l'Homebrew Launcher
1. Lancez WUP Installer GX2
1. Select the Homebrew Channel
1. Touchez "Install", puis "Yes" pour confirmer
1. Choisissez "NAND" comme destination
1. Select WUP Installer GX2
1. Touchez "Install", puis "Yes" pour confirmer
1. If you plan to install games to a USB HDD, select "USB" as the destination; if you do not plan to use a USB HDD, select "NAND" as the destination
1. Appuyez sur le bouton (Home) puis quittez le logiciel Éditeur Mii quand l'installation est terminée

___

Vous pouvez à présent lancer le Homebrew Launcher à partir de la SysNAND patché en lançant la Chaîne Homebrew.
{: .notice}

Vous pouvez à présent lancer le WUP Installer GX2 sur la SysNAND patchée en lançant le logiciel depuis le menu Home.
{: .notice}

Pour plus de sécurité, vous *devriez* faire une sauvegarde de la NAND de votre Wii U en suivant le guide [Sauvegarde NAND](nand-backup).
{: .notice--primary}

Notez que pour accéder à l’eShop avec les adresses DNS du bloquage des mise à jours, vous devez d’abord exécuter NNU-Patcher via le l'Homebrew Launcher (NNU-Patcher est un patch temporaire et devra être lancé à nouveau si vous avez besoin d’accéder à l’eShop après un redémarrage).
{: .notice--info}

Notez que, dû à un bug, WUP Installer GX2 peut seulement installer sur un disque USB (comme un HDD USB) si vous déplacez WUP Installer GX2 sur un stockage USB en utilisant le Gestionnaire de Données dans les Paramètres Systèmes.
{: .notice--warning}

Pour toute information concernant la copie de vos disques afin de pouvoir les installer sur votre console et jouer sans le disque, consultez la page de [disc2app](disc2app).
{: .notice--success}

Pour toute information concernant l'installation de « mods » vWii sur la SysNAND, consultez la page [Mod de la vWii](vwii-modding).
{: .notice--success}

Pour toute information concernant la désinstallation du CFW en toute sécurité et un retour aux paramètres d'usine, consultez la page [Désinstallation du CFW](uninstall-cfw).
{: .notice--warning}
