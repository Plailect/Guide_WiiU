---
title: "Homebrew Launcher"
---

Le Homebrew Launcher est une application homebrew qui répertorie et permet le lancement d’autres applications homebrew de la carte SD.
{: .notice}

La méthode de lancement de HBL nécessite que votre Wii U soit raccordée à internet, puisque nous ferons usage du navigateur intégré.
{: .notice--info}

##### Instructions

1. Lancer le navigateur sur votre Wii U
1. Allez dans les paramètres de navigation et cliquez sur "Mettre les paramètres à zéro"
  + Cette démarche permet une meilleur compatibilité avec les exploits utilisés dans le navigateur
  + En contre-partie, cela supprimera aussi tout vos favoris ainsi que votre historique et vos paramètres de navigation
1. Retournez dans le navigateur

### For 5.5.0 & 5.5.1

1. Ouvrez le site `http://loadiine.ovh`
  + Vous pouvez sauvegarder cette adresse dans vos favoris afin de gagner du temps lors des prochaines sessions
1. Ensure that the drop down box shows "Homebrew Launcher 1.4 (5.5.0 - 5.5.1)", then press "Submit"
  + Plusieurs tentatives peuvent être nécessaires pour lancer le hack
  + Dans certaines situations, l'option "Homebrew Launcher 1.3 (5.3.2 - 5.4.0 - 5.5.0 - 5.5.1)" peut mieux marcher pour lancer le Homebrew Launcher
  + Si votre console freeze (ne répond plus), vous pouvez forcer l'extinction de celle-ci en maintenant le bouton Power et réessayer
1. Votre console devrait maintenant charger le Homebrew Launcher

### For 5.5.2

1. Go to `http://u.drg.li/`
  + Vous pouvez sauvegarder cette adresse dans vos favoris afin de gagner du temps lors des prochaines sessions
2. Click "2"
  + Plusieurs tentatives peuvent être nécessaires pour lancer le hack
  + In some situations, the other options/numbers are more reliable.
  + Si votre console freeze (ne répond plus), vous pouvez forcer l'extinction de celle-ci en maintenant le bouton Power et réessayer
1. Votre console devrait maintenant charger le Homebrew Launcher
