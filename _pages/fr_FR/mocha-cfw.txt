### Lecture requise

Mocha CFW est un custom firmware qui patche le contrôle de signatures, la vérification de la région et autorise d'autres fonctionnalités de custom firmware.

Afin d'utiliser Mocha CFW, vous devez entrer dans le Homebrew Launcher via le navigateur web et lancer Mocha CFW à chaque fois que votre console redémarre.

## Ce dont vous avez besoin

+ Le fichier personnalisé `config.ini` fourni dans [Pour Commencer](get-started) doit déjà se trouver dans le répertoire `/wiiu/apps/mocha/` sur votre carte SD

### Instructions

1. Lancez Mocha CFW
1. La console va retourner dans le Homebrew Launcher et autoriser les fonctionnalités de custom firmware
  + Ceci permettra aux titres non signés (par exemple le Homebrew Channel) d'être lancés directement depuis le menu Home jusqu'au prochain redémarrage

___

Notez que vous devrez lancer Mocha CFW depuis le Homebrew Launcher à chaque fois que vous redémarrez afin de ré-autoriser les fonctionnalités de custom firmware.
