title: "FAQ"
---

<a name="faq_latestfw" />**D:** *Ho installato l'ultima versione di sistema, posso comunque installare un custom firmware?*    
**A:** Si

<a name="faq_oldfw" />**D:** *Ho una Wii U alla versione X.X.X. What can / should I do with it?*    
**A:** This guide is only compatible with versions 5.5.0 to 5.5.3. Versioni precedenti non sono al momento supportate.

<a name="faq_updates" />**D:** *Ora che Nintendo ha smesso di produrre Wii U, non ci saranno più aggiornamenti?*    
**R:** No! The versions 5.5.2 and 5.5.3 released well after the end of production. This is why you must always ensure that you have set DNS servers that will block updates.

<a name="faq_update" />**Q:** *Is it safe to update a Wii U on CFW?*    
**R:** No! Assicurati di aver sempre impostati i server DNS che bloccano gli aggiornamenti.

<a name="faq_updatemocha" />**D:** *Come aggiorno Mocha CFW all'ultima versione?*    
**R:** Scarica l'ultima versione di [Mocha CFW](https://github.com/dimok789/mocha/releases/latest), quindi copia il file `mocha.elf` dall'archivio `.zip` di Mocha CFW nella cartella `/wiiu/apps/mocha/` della tua scheda SD (sovrascrivi il file esistente)

<a name="faq_risky" />**D:** *Quanto è rischioso modificare la mia console?*    
**R:** I brick sono estremamente rari e sono di solito causati dagli utenti che saltano le istruzioni o che non le seguono.

<a name="faq_homebrew" />**D:** *Potrò avviare emulatori ed homebrew fantastici al termine della guida?*    
**R:** Sì! Questa guida non solo ti permette di avviare l'Homebrew Launcher, ma le funzionalità aggiuntive permangono anche dopo un aggiornamento di sistema.

<a name="faq_regionfree" />**D:** *Potrò giocare a giochi comprati all'estero?*    
**R:** Sì! CFW supports Region Free patches, which enables users to install games from any region without any extra modification or steps.

<a name="faq_support" />**D:** *Dove posso andare se dovessi aver bisogno di ulteriore supporto?*    
**R:** Se hai bisogno di aiuto, chiedi pure su [Nintendo Homebrew su Discord](https://discord.gg/MWxPgEp) (in inglese).

<a name="faq_sdsize" />**D:** *Quanto grande deve essere la mia scheda SD?*    
**A:** Your SD card is recommended to be at least 16GB or 32GB to have enough space to dump and install games. If you want to do a full NAND backup of a white (8GB) Wii U model, your SD card needs to be at least 16GB in size. If you want to do a full NAND backup of a black (32GB) Wii U model, your SD card needs to be at least 64GB in size.

<a name="faq_NNID" />**D:** *Posso mantenere il mio NNID?*    
**R:** Se parti con un NNID assegnato e segui correttamente ogni passaggio, al termine della guida avrai mantenuto il tuo NNID originale.

<a name="faq_nopc" />**D:** *Posso seguire questa guida senza un computer a disposizione (ad esempio, da un telefono Android)?*    
**R:** Sì, a patto che tu abbia modo di trasferire file sulla scheda SD!
