---
title: "Homebrew Launcher (Channel)"
---

Isto irá instalar o Homebrew Channel para que, uma vez você possua uma SysNAND alterada em execução, o Homebrew Launcher aparecerá como um ícone no menu do sistema.
{: .notice--info}

#### Instruções

1. Enter the Homebrew Launcher using `http://loadiine.ovh`, Haxchi, or Coldboot Haxchi
  + Note that Haxchi users must launch Haxchi once to patch SysNAND, then run it a second time while holding (A) to launch the Homebrew Launcher
1. **Homebrew Launcher Users Only:** Launch Mocha CFW, then relaunch the Homebrew Launcher
1. Launch WUP Installer GX2
1. Select the Homebrew Channel and WUP Installer GX2
1. Press "Install", then press "Yes" to confirm
1. Select "NAND" as the destination
1. Press (Home), then close the software to exit the installer once it has completed installing

___

Você pode iniciar o Homebrew Launcher na SysNAND alterada apenas iniciando o Homebrew Channel.
{: .notice}

You can now launch WUP Installer GX2 on patched SysNAND just by launching it from the system menu.
{: .notice}

For increased safety, you *should* create a NAND backup of your device by following [NAND Backup](nand-backup).
{: .notice--primary}

Note que para acessar o eShop com os endereços DNS de bloqueio de atualizações definidos, você deve primeiro iniciar o NNU-Patcher através do Homebrew Launcher (NNU-Patcher é uma alteração temporária e será necessário rodá-lo novamente se você desejar acessar o eShop após uma reinicialização).
{: .notice--info}

Note that, due to a bug, WUP Installer GX2 can only install to USB drives (such as a USB HDD) if you move WUP Installer GX2 itself to USB storage using the Data Management menu of the System Settings.
{: .notice--warning}

Para informações sobre como descarregar seus discos para um formato instalável, para jogá-los sem a necessidade do disco, confira a página [disc2app](disc2app).
{: .notice--success}

Para informações sobre como installar mods de vWii na SysNAND, confira a página [Modding do vWii](vwii-modding).
{: .notice--success}

Para informações sobre como desinstalar o CFW com segurança e retornar ao modo de fábrica, confira a página [Desinstalar o Mocha CFW](uninstall-mocha-cfw).
{: .notice--warning}
