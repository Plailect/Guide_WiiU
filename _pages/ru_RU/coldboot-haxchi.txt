### Обязательно к прочтению

CBHC (Coldboot Haxchi) изменяет системный тайл, запускающийся по умолчанию при старте SysNAND, на игру для DS virtual console, которая затем запускает пропатченный SysNAND.

Выполнение инструкций на этой странице не обязательно. Обычный Haxchi прекрасно работает; станет просто слегка удобней. Если вы хотите пропустить установку Coldboot Haxchi, переходите к [Homebrew Launcher (Channel)](homebrew-launcher-(channel)).
{: .notice--primary}

Для продолжения необходимо иметь функционирующий Haxchi. Без него получите гарантированный БРИК!

Ваша игра для DS virtual console, используемая для Haxchi, ДОЛЖНА находится во внутренней памяти Wii U. Если она находится на USB-носителе, гарантированно получите БРИК!

Ваша игра для DS virtual console, используемая для Haxchi, ДОЛЖНА быть ЛЕГАЛЬНОЙ копией, в иных случаях вы гарантированно получите БРИК!

Все USB-устройства должны быть отключены перед установкой, иначе есть шанс получить БРИК!

### Инструкция

1. Отключите все USB-устройства от Wii U
1. Войдите в Homebrew Launcher, удерживая (A) во время запуска игры для DS virtual console с установленным Haxchi
1. Запустите CBHC installer
1. Выберите игру для DS virtual console, в которую вы собираетесь установить CBHC, затем нажмите (A) для подтверждения
  + **Игра уже ДОЛЖНА содержать установленный и работающий Haxchi**
1. Ознакомьтесь с появившимся предупреждением и нажмите (A) для установки
1. После окончания процесса вы вернетесь в системное меню
1. Вы можете подсоединить обратно все USB-устройства, которые были отключены раннее.
1. Перезагрузите Wii U
1. Если эксплойт сработал корректно, вы увидите меню с различными опциями загрузки
1. Hover over **"Autoboot: Disabled"**, then press (A) on it until it says **"Autoboot: System Menu"**
1. Press (A) on "Boot System Menu"
1. Reboot your Wii U to ensure the exploit works automatically as intended
1. Move the CBHC DS virtual console game to the last page of the system menu
  + Read the warnings at the bottom of this page carefully to avoid bricking your device!

___

Your device should now automatically launch into Coldboot Haxchi patched SysNAND on boot.
{: .notice}

{% capture notice-1 %}
You can press (Home) while CBHC is booting to access the CBHC boot options menu. This is what each of the boot options do:

  + Boot System Menu -> Boots into SysNAND
  + Boot Homebrew Launcher -> Boots into the Homebrew Launcher on SysNAND
  + Boot Mocha CFW -> Boots into Mocha CFW with RedNAND
  + Boot fw.img on SD Card -> Boots into any older fw.img based CFW from the SD card
  + Boot vWii System Menu -> Boots into vWii on SysNAND
  + Boot vWii Homebrew Channel -> Boots into vWii Homebrew Channel on SysNAND

{% endcapture %}
