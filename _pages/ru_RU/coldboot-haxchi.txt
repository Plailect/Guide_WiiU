### Обязательно к прочтению

CBHC (Coldboot Haxchi) changes your default system title to make your device launch the DS virtual console game by default instead of the system menu, which will then load custom firmware.

Выполнение инструкций на этой странице не обязательно. Обычный Haxchi прекрасно работает; станет просто слегка удобней. Если вы хотите пропустить установку Coldboot Haxchi, переходите к [Homebrew Launcher (Channel)](homebrew-launcher-(channel)).
{: .notice--primary}

{% capture notice-1 %}
Для продолжения необходимо иметь функционирующий Haxchi. Без него получите гарантированный БРИК!

Ваша игра для DS virtual console, используемая для Haxchi, ДОЛЖНА находится во внутренней памяти Wii U. Если она находится на USB-носителе, гарантированно получите БРИК!

Ваша игра для DS virtual console, используемая для Haxchi, ДОЛЖНА быть ЛЕГАЛЬНОЙ копией, в иных случаях вы гарантированно получите БРИК!
{% endcapture %}

### Инструкция

1. Отключите все USB-устройства от Wii U
1. Войдите в Homebrew Launcher, удерживая (A) во время запуска игры для DS virtual console с установленным Haxchi
1. Запустите CBHC installer
1. Выберите игру для DS virtual console, в которую вы собираетесь установить CBHC, затем нажмите (A) для подтверждения
  + **Игра уже ДОЛЖНА содержать установленный и работающий Haxchi**
1. Ознакомьтесь с появившимся предупреждением и нажмите (A) для установки
1. После окончания процесса вы вернетесь в системное меню
1. Вы можете подсоединить обратно все USB-устройства, которые были отключены раннее.
1. Перезагрузите Wii U
1. Если эксплойт сработал корректно, вы увидите меню с различными опциями загрузки
1. Наведите курсор на **"Autoboot: Disabled"** и нажимайте (A) до тех пор, пока надпись не поменяется на **"Autoboot: System Menu"**
1. Нажмите (A) на пункте "Boot System Menu"
1. Перезагрузите консоль, чтобы убедиться, что эксплойт срабатывает автоматически в соответствии с настройками
1. Переместите игру для DS virtual console с установленным CBHC на последнюю страницу системного меню
  + Внимательно прочитайте предупреждения внизу этой страницы. Это поможет вам уберечь консоль от брика!

___

Your device should now automatically launch into Coldboot Haxchi custom firmware on boot.
{: .notice}

{% capture notice-1 %}
Нажатие кнопки (Home) во время загрузки CBHC вызовет меню опций загрузки. Ниже краткое описание всех опций:

  + Boot System Menu -> Boots into custom firmware
  + Boot Homebrew Launcher -> Boots into the Homebrew Launcher
  + Boot Mocha CFW -> Boots into Mocha CFW (no reason to use this with CBHC)
  + Boot fw.img on SD Card -> Загрузить старую CFW из файла fw.img из корня SD-карты
  + Boot vWii System Menu -> Boots into vWii
  + Boot vWii Homebrew Channel -> Boots into vWii Homebrew Channel

{% endcapture %}

**Никогда не удаляйте пользовательский NNID аккаунт, через который была куплена игра для DS virtual console с установленным CBHC, иначе получите БРИК!**
{: .notice--danger}
