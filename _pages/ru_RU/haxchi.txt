### Обязательно к прочтению

Haxchi - это кастомная прошивка, которая патчит проверку подписи, проверку региона и запускает различный пользовательский код на фоне работающей прошивки.

Haxchi - это эксплойт, который позволяет нам запускать CFW напрямую из системного меню без использования Homebrew Launcher.

### Что понадобится

+ Купленная копия одной из совместимых с эксплойтом игр для DS virtual console, установленная во внутреннюю память Wii U
  + Если у вас еще нет совместимой с эксплойтом игры, вернитесь к [Началу](get-started)

### Инструкция

1. Запустите Haxchi Installer
1. Выберите игру для DS virtual console, в которую вы собираетесь установить Haxchi, затем нажмите (A) для подтверждения
1. Ознакомьтесь с появившимся предупреждением и нажмите (A) для установки
1. После окончания процесса, запустите вашу игру для DS virtual console (будет называться "Haxchi")
1. Консоль перезагрузится в пропатченный SysNAND
  + Вы сможете запускать неподписанные приложения (например homebrew channel или резервные копии игр) прямо из системного меню

___

В данный момент для загрузки пропатченного SysNAND вам после каждой перезагрузки нужно запускать игру для DS virtual console. На следующей странице мы установим CBHC (Coldboot Haxchi), который автоматизирует этот процесс.
{: .notice--info}

**Вы можете удерживать различные кнопки при запуске игры для DS virtual console чтобы изменить опции загрузки. Ниже краткое описание всех опций:**

  + Нет -> Загрузка в пропатченный SysNAND
  + (A) -> Загрузка в Homebrew Launcher
