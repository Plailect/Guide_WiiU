### Обязательно к прочтению

Haxchi - это эксплойт, который позволяет нам запускать CFW напрямую из системного меню без использования Homebrew Launcher.

Он также содержит кастомную прошивку, которая патчит проверку подписи, проверку региона и запускает различный пользовательский код на фоне работающей прошивки.

### Что понадобится

+ Купленная копия одной из совместимых с эксплойтом игр для DS virtual console, установленная во внутреннюю память Wii U
  + Если у вас еще нет совместимой с эксплойтом игры, вернитесь к [Началу](get-started)
+ Измененный файл `config.txt` со страницы [Начало](get-started) уже должен быть в папке `/haxchi/` на SD-карте (*не* тот файл, который присутствует в Haxchi по умолчанию)
  + Если вы используете файл конфигурации по умолчанию (из `.zip-архива` Haxchi), то Haxchi будет работать не так, как ожидается
  
### Инструкция

1. Запустите Haxchi Installer
1. Выберите игру для DS virtual console, в которую вы собираетесь установить Haxchi, затем нажмите (A) для подтверждения
1. Ознакомьтесь с появившимся предупреждением и нажмите (A) для установки
1. После окончания процесса, запустите вашу игру для DS virtual console (будет называться "Haxchi")
1. Консоль перезагрузится в кастомную прошивку
  + Вы сможете запускать неподписанные приложения (например Homebrew Launcher) прямо из системного меню

___

В данный момент для загрузки кастомной прошивки вам после каждой перезагрузки нужно запускать игру для DS virtual console. На следующей странице опционально устанавливается Coldboot Haxchi, который может автоматизировать этот процесс.

### Методы

___

#### Stay on Standard Haxchi

Standard Haxchi is the exploit that has already been installed on your device.

This exploit comes with no risk of damaging your device, but it is less convenient as you must perform the Haxchi exploit manually after each reboot.

### Continue to [Hombrew Launcher (Channel)](homebrew-launcher-(channel)).
{: .notice--primary}

___

#### Upgrade to Coldboot Haxchi

CBHC (Coldboot Haxchi) изменяет системный тайтл, запускающийся по умолчанию при старте консоли, с системного меню на игру для DS virtual console. Это заставляет консоль автоматически запускать кастомную прошивку при загрузке системы.

Этот эксплойт чуть более опасен чем стандартный Haxchi, поскольку эксплойт в игре для DS Virtual Console является *единственным* способом загрузки системы. Если игра когда-либо будет удалена или сам эксплойт будет удален, то установка CBHC приведет к БРИКУ!
