### Обязательно к прочтению

Мы установим Homebrew Channel таким образом, что при запуске пропатченного SysNAND, Homebrew Launcher появится в виде иконки в системном меню.

### Инструкция

1. Запустите Homebrew Launcher
  + Пользователи CBHC должны нажать кнопку (Home) во время загрузки CBHC для доступа к меню опций загрузки, где необходимо выбрать Homebrew Launcher
  + Пользователи Haxchi должны сначала запустить Haxchi, чтобы пропатчить SysNAND, а затем запустить его еще раз, удерживая (A), чтобы запустить Homebrew Launcher
  + Пользователи Mocha CFW должны сперва запустить Mocha CFW, а затем перезапустить Homebrew Launcher
1. Запустите WUP Installer GX2
  + Если WUP Installer GX2 падает с ошибкой "DSi Exception Has Occurred", перезапустите консоль и попробуйте заново.
1. Выберите Homebrew Channel
1. Нажмите "Install" для установки, затем "Yes" для подтверждения
1. Выберите "NAND" в качестве места установки
1. Выберите WUP Installer GX2
1. Нажмите "Install" для установки, затем "Yes" для подтверждения
1. Если вы планируете устанавливать игры на USB HDD, выберите "USB" в качестве места установки; если вы не планируете использовать USB HDD, выберите "NAND"
1. Нажмите (Home) и выберите Закрыть программу, чтобы выйти из установщика после завершения установки

___

Теперь вы можете запускать Homebrew Launcher из пропатченного SysNAND, просто воспользовавшись Homebrew Channel.
{: .notice}

Теперь вы можете запускать WUP Installer GX2 из пропатченного SysNAND, используя иконку в системном меню.
{: .notice}

В целях безопасности крайне *желательно* выполнить резервное копирование NAND, следуя [инструкции](nand-backup).
{: .notice--primary}

Помните, что для доступа к eShop с блокирующими обновления адресами DNS, вам необходимо сначала запустить NNU-Patcher из Homebrew Launcher (NNU-Patcher - временный патч, и его нужно запускать после каждой перезагрузки, если вам нужен доступ к eShop).
{: .notice--info}

Note that, due to a bug, WUP Installer GX2 can only install to USB drives (such as a USB HDD) if you move WUP Installer GX2 itself to USB storage using the Data Management menu of the System Settings.
{: .notice--warning}
