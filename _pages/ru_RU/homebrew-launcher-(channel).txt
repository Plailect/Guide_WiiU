### Обязательно к прочтению

Мы установим Homebrew Channel таким образом, что при запуске кастомной прошивки, Homebrew Launcher появится в виде иконки в системном меню.

### Инструкция

#### Часть I - Установка Homebrew Launcher Channel

1. Запустите Homebrew Launcher
  + Пользователи CBHC должны нажать кнопку (Home) во время загрузки CBHC для доступа к меню опций загрузки, где необходимо выбрать Homebrew Launcher
  + Пользователи Haxchi должны сначала запустить Haxchi, чтобы включить кастомную прошивку, а затем запустить его еще раз, удерживая (A), чтобы запустить Homebrew Launcher
  + Пользователи Mocha CFW уже должны находиться в Homebrew Launcher
1. Запустите WUP Installer GX2
  + Если WUP Installer GX2 падает с ошибкой "DSi Exception Has Occurred", перезапустите консоль и попробуйте заново.
1. Выберите Homebrew Channel
1. Нажмите "Install" для установки, затем "Yes" для подтверждения
1. Выберите "NAND" в качестве места установки
1. Выберите WUP Installer GX2
1. Нажмите "Install" для установки, затем "Yes" для подтверждения
1. Выберите "NAND" в качестве места установки
1. Нажмите (Home) и выберите Закрыть программу, чтобы выйти из установщика после завершения установки

#### Часть II - Блокировка обновления системы

Обратите внимание, что для доступа к eShop с этими адресами DNS вам необходимо сначала запустить NNU-Patcher из Homebrew Launcher. Помните также, что NNU-Patcher - временный патч, который необходимо запускать после каждой перезагрузки, если вам нужен доступ к eShop.

Обратите внимание, что если вы не заблокируете обновления этим методом на текущем и всех будущих интернет-соединениях, то все обновления для Wii U будут скачиваться и устанавливаться *автоматически* без вашего ведома и *не могут* быть отключены. 
{: .notice--danger}

1. Перейдите в Системные настройки (System Settings), Интернет (Internet), Подключение к Интернету (Connect to the Internet), нажмите (X) для того, чтобы отобразить имеющиеся подключения
1. Для каждого из имеющихся подключений (и для всех созданных в будущем) выполните следующие инструкции
  + Выберите подключение
  + Выберите "Изменить настройки" (Change Settings)
  + Перейдите на вторую страницу и выберите "DNS"
  + В пункте меню DNS выберите "Не получать автоматически" (Don't Auto-Obtain)
  + Введите указанные ниже адреса DNS
  + {:.bigdns} `168.235.092.108`
  + {:.bigdns} `081.004.127.020`
  + Нажмите "Подтвердить" (Confirm), а затем (B), чтобы сохранить внесенные изменения
  + Эти сервера заблокируют получение обновлений на вашей консоли

___

Теперь вы можете запускать Homebrew Launcher на кастомной прошивке, просто воспользовавшись Homebrew Channel.
{: .notice--info}

Теперь вы можете запускать WUP Installer GX2 на кастомной прошивке, используя иконку в системном меню.
{: .notice--info}

В целях безопасности крайне *желательно* выполнить резервное копирование NAND, следуя [инструкции](nand-backup).
{: .notice--primary}

Помните, что для доступа к eShop с блокирующими обновления адресами DNS, вам необходимо сначала запустить NNU-Patcher из Homebrew Launcher (NNU-Patcher - временный патч, и его нужно запускать после каждой перезагрузки, если вам нужен доступ к eShop).
{: .notice--info}

Для информации о том, как сохранять игры с дисков в формате, пригодном для установки в систему для последующей игры без диска, перейдите к странице [disc2app](disc2app).
{: .notice--success}

Для информации об установке модов vWii на кастомной прошивке, перейдите к странице [Модификация vWii](vwii-modding).
{: .notice--success}

Для информации о безопасном удалении кастомной прошивки и возвращении консоли к заводскому состоянию, перейдите на страницу [Удаление кастомной прошивки](uninstall-cfw).
{: .notice--warning}
