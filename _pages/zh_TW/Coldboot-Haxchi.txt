CBHC (Coldboot Haxchi) changes your default system title to make your SysNAND launch the DS virtual console game by default instead of the system menu, which will then load patched SysNAND.
{: .notice}

本節為選用章節。 Regular Haxchi works just fine as is; this just offers extra convenience. If you would like to skip Haxchi, continue to [Homebrew Launcher (Channel)](homebrew-launcher-(channel)).
{: .notice--info}

**你 _必須_ 先裝好 Haxchi 才能繼續。 否則－保。證。磚。機！**
{: .notice--danger}

**你的 Haxchi DS Virtual Console 遊戲 _必須_ 先裝在 Wii U 內存中才能繼續。如果是存在 USB 外接裝置上的話－保。證。磚。機！**
{: .notice--danger}

**你的 Haxchi DS Virtual Console 遊戲 _必須_ 是正版的，否則－保。證。磚。機！**
{: .notice--danger}

#### 操作說明

1. Enter the Homebrew Launcher by holding (A) while launching your Haxchi DS virtual console game
1. 啟動 CBHC installer
1. 選擇你要安裝 CBHC 的 DS Virtual Console 遊戲，並按下「A」鍵以選取
  + **所選的遊戲 _必須_ 要先裝好 Haxchi 才能繼續**
1. 讀完警示，並按下「A」鍵安裝
1. Once this is complete, you should be back at the system menu
1. 重新啟動你的 Wii U
1. 如果漏洞順利執行，你將會看到各種選項的開機選單
1. Hover over **"Autoboot: Disabled"**, then press (A) on it until it says **"Autoboot: System Menu"**
1. Press (A) on "Boot System Menu"
1. 重啟你的 Wii U 以確保漏洞仍能自動執行
1. Move the CBHC DS virtual console game to the last page of the system menu
  + Read the warnings at the bottom of this page carefully to avoid bricking your device!

___

Your device should now automatically launch into Coldboot Haxchi patched SysNAND on boot.
{: .notice}

{% capture notice-1 %}
**你可以在 CBHC 啟動時按下「Home」鍵進入 CBHC 開機選單。 每個選項的功能如下：**

    + Boot System Menu -> 開機至 SysNAND
    + Boot Homebrew Launcher -> 開機至 SysNAND 的 Homebrew Launcher 中
    + Boot Mocha CFW -> 開機至 RedNAND 的 Mocha CFW 中
    + Boot fw.img on SD Card -> 透過 SD 卡中任一舊式 fw.img 鏡像檔開機
    + Boot vWii System Menu -> 開機至 SysNAND 的 vWii 中
    + Boot vWii Homebrew Channel -> 開機至 SysNAND vWii 的 Homebrew Channel 中

{% endcapture %}
