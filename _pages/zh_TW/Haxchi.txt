Haxchi is a custom firmware that patches signature checks, region checks, and runs other custom code in the background.
{: .notice}

Haxchi is an exploit that allows us to launch Mocha CFW directly from the system menu without using the Homebrew Launcher.
{: .notice--info}

If you need to purchase one of the exploitable games, run NNU-Patcher from the Homebrew Launcher then launch the eShop. 一旦購買完畢後，請繼續閱讀 [Homebrew Launcher](homebrew-launcher)。
{: .notice--primary}


#### 必備項目

+ 下列任一 DS 虛擬主機 遊戲（須為正版且安裝於 Wii U 內存中）
  + **千萬不要**安裝遊戲在 USB 儲存裝置中
  + 目前為止，《東北大學未來科學技術共同研究中心川島隆太教授監修 大人的DS腦力鍛鍊》是 eShop 上最便宜的遊戲 ($6.99 美元)
  + 如果你先前已經安裝過了上述遊戲，請將它刪除並再重新透過 eShop 下載。 舊版的遊戲無法與 Haxchi 相容
  + 注意未來你將無法再遊玩你所選的遊戲
  + **你的 Haxchi DS 虛擬主機 遊戲 _必須_ 是正版的！**

#### 操作說明

1. 啟動 Haxchi installer
1. 選擇你要安裝 Haxchi 的 DS 虛擬主機 遊戲，並按下「A」鍵以選取
1. 讀完警示，並按下「A」鍵安裝
1. 安裝完畢後，啟動你 Haxchi 改後的 DS 虛擬主機 遊戲（叫「Haxchi」）
1. The console will reboot into patched SysNAND
  + This will allow unsigned titles (such as the homebrew channel and game backups) to be launched directly from the system menu
