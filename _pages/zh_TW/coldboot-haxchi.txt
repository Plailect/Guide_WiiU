### 必讀事項

CBHC (Coldboot Haxchi) changes your default system title to make your device launch the DS virtual console game by default instead of the system menu, which will then load custom firmware.

本節為選用章節。 普通的 Haxchi 就已經很好了；只是此步驟會讓啟動它更方便些。 如果您想跳過 冷起 Haxchi，請繼續至 [Homebrew Launcher (Channel)](homebrew-launcher-(channel))。

### 操作說明

1. 拔除所有連接至 Wii U 的 USB 主機
1. 在啟動 Haxchi DS 虛擬主機遊戲時，按下「A」進入 Homebrew Launcher
1. 啟動 CBHC installer
1. 選擇您要安裝 CBHC 的 DS 虛擬主機 遊戲，並按下「A」鍵以選取
  + **所選的遊戲 _必須_ 要先裝好 Haxchi 才能繼續**
1. 讀完警示，並按下「A」鍵安裝
1. 完成後，您將會回到系統主選單中
1. 您現在可以將剛剛拔除的 USB 主機插回了。
1. 重新啟動您的 Wii U
1. 如果漏洞順利執行，您將會看到各種選項的開機選單
1. 將游標移至**「Autoboot: Disabled」**上，並按「A」鍵直到選項變為**「Autoboot: System Menu」**為止
1. 在「Boot System Menu」上按「A」
1. 重啟您的 Wii U 以確保漏洞仍能自動執行
1. 將 CBHC 改後的 DS 虛擬主機遊戲移到系統選單的最後一頁
  + 請仔細閱讀本頁底部的警告，避免將您的主機變磚！

___

Your device should now automatically launch into Coldboot Haxchi custom firmware on boot.
{: .notice}

{% capture notice-1 %}
您可以在 CBHC 啟動時按下「Home」鍵進入 CBHC 開機選單。 每個選項的功能如下：

  + Boot System Menu -> Boots into custom firmware
  + Boot Homebrew Launcher -> Boots into the Homebrew Launcher
  + Boot Mocha CFW -> Boots into Mocha CFW (no reason to use this with CBHC)
  + Boot fw.img on SD Card -> 透過 SD 卡中任一舊式 fw.img 鏡像檔開機
  + Boot vWii System Menu -> Boots into vWii
  + Boot vWii Homebrew Channel -> Boots into vWii Homebrew Channel

{% endcapture %}
