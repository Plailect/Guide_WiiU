### Required Reading

CBHC (Coldboot Haxchi) 將把您的預設系統 title 修改為自動啟動修改後的 DS 虛擬主機 遊戲，讓您自動開機至修改後的 SysNAND 中，而非普通的系統選單。

本節為選用章節。 普通的 Haxchi 就已經很好了；只是此步驟會讓啟動它更方便些。 If you would like to skip Coldboot Haxchi, continue to [Homebrew Launcher (Channel)](homebrew-launcher-(channel)).
{: .notice--primary}

{% capture notice-1 %}
You MUST have a functional Haxchi setup for this to work. If you do not, you will BRICK!

Your Haxchi DS virtual console game MUST be on the internal memory of your Wii U. If it is on a USB drive, you will BRICK!

Your Haxchi DS virtual console game MUST be a LEGITIMATE copy or you will BRICK!

You must remove all USB devices before installing or you may BRICK!
{% endcapture %}

### Instructions

1. 拔除所有連接至 Wii U 的 USB 裝置
1. 在啟動 Haxchi DS 虛擬主機遊戲時，按下「A」進入 Homebrew Launcher
1. 啟動 CBHC installer
1. 選擇您要安裝 CBHC 的 DS 虛擬主機 遊戲，並按下「A」鍵以選取
  + **所選的遊戲 _必須_ 要先裝好 Haxchi 才能繼續**
1. 讀完警示，並按下「A」鍵安裝
1. 完成後，您將會回到系統主選單中
1. 您現在可以將剛剛拔除的 USB 裝置插回了。
1. 重新啟動您的 Wii U
1. 如果漏洞順利執行，您將會看到各種選項的開機選單
1. 將游標移至**「Autoboot: Disabled」**上，並按「A」鍵直到選項變為**「Autoboot: System Menu」**為止
1. 在「Boot System Menu」上按「A」
1. 重啟您的 Wii U 以確保漏洞仍能自動執行
1. 將 CBHC 改後的 DS 虛擬主機遊戲移到系統選單的最後一頁
  + 請仔細閱讀本頁底部的警告，避免將您的裝置變磚！

___

您的裝置現在應該在啟動時都會自動啟動修改後的 SysNAND。
{: .notice}

{% capture notice-1 %}
您可以在 CBHC 啟動時按下「Home」鍵進入 CBHC 開機選單。 每個選項的功能如下：

  + Boot System Menu -> 開機至 SysNAND
  + Boot Homebrew Launcher -> 開機至 SysNAND 的 Homebrew Launcher 中
  + Boot Mocha CFW -> 開機至 RedNAND 的 Mocha CFW 中
  + Boot fw.img on SD Card -> 透過 SD 卡中任一舊式 fw.img 鏡像檔開機
  + Boot vWii System Menu -> 開機至 SysNAND 的 vWii 中
  + Boot vWii Homebrew Channel -> 開機至 SysNAND vWii 的 Homebrew Channel 中

{% endcapture %}
